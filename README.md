# SafeReady

SafeReady is my mobile disaster preparedness final project app. The app is designed for everyday users and focuses on personal preparedness before emergencies happen, rather than live disaster prediction or professional emergency response.

## Project Overview

The aim of this project is to support users in building practical preparedness habits through guided activities and simple safety tools.

The app includes:
- onboarding
- home dashboard
- tasks screen
- alerts screen
- resources screen
- profile screen
- safety kit
- evacuation plan
- emergency contacts
- checklist
- quiz
- settings
- emergency panel with emergency call shortcuts and location sharing
- points, score, badges, and progress summary

## Technologies Used

This project was built with:
- React Native
- Expo
- Expo Router
- AsyncStorage
- TypeScript

## Running the Project

1. Install dependencies

```bash
npm install
```

2. Start the Expo development server

```bash
npx expo start
```

3. Run the app on:
- an Android emulator
- an iOS simulator
- or the Expo Go app on a physical device
